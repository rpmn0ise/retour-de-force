## Description

Un circuit touge de 8km inspiré des routes de montagne japonaises.
Conçu pour le drift technique : enchaînements de virages serrés, dévers variés,
surface asphaltée réaliste avec grip évolutif selon la météo.

## Caractéristiques

- **Longueur** : 8.2 km aller simple
- **Dénivelé** : +420m / -420m
- **Variantes** : Jour, Nuit, Pluie, Brouillard
- **Spawn points** : 3 (sommet, mi-pente, bas)
- Compatible multijoueur

## Installation

1. Télécharger et extraire le `.zip`
2. Placer le dossier dans `BeamNG/levels/`
3. La map apparaît dans le sélecteur de scène
