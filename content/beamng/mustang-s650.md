## Description

Ce mod est plein de technologie du monde réel, avec plus de 72 déclencheurs interactables à l'intérieur seul! Et un système d'infodivertissement Mustang natif entièrement fonctionnel, Working Apple Carplay, Working Radio, Subwoofers et bien plus encore!

## Installation

1. Télécharger le fichier `.zip`
2. Ouvrir **BeamNG.drive** → aller dans le **Repository** (icône carton)
3. Glisser-déposer le fichier `.zip` dans la fenêtre
4. Activer le mod dans la liste, puis redémarrer le jeu

> Compatible BeamNG.drive 0.30+
