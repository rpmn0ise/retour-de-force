## Description

BMW M8 F92 by KingMods!  High-quality mod BMW  In addition to the standard suspension, there are 2 more suspension types M8 and M8 Competition, which differ from each other in stiffness, ground clearance and more!   as well as the M8 and M8 Competition nameplates!

## Installation

1. Télécharger le fichier `.zip`
2. Ouvrir **BeamNG.drive** → aller dans le **Repository** (icône carton)
3. Glisser-déposer le fichier `.zip` dans la fenêtre
4. Activer le mod dans la liste, puis redémarrer le jeu

> Compatible BeamNG.drive 0.30+
