## Description

Le mod de trafic réaliste simplifié de l'agent - une refonte complète du trafic pour BeamNG qui remplace le système de trafic standard par une solution plus réaliste, variée et optimisée pour les performances. Ce mod est conçu pour apporter des scénarios de trafic dynamique et immersif au jeu tout en améliorant le réalisme visuel et l'efficacité.

## Installation

1. Télécharger le fichier `.zip`
2. Ouvrir **BeamNG.drive** → **Repository** (icône carton)
3. Glisser-déposer le `.zip` dans la fenêtre
4. Activer le mod puis redémarrer le jeu

> Compatible BeamNG.drive 0.30+
