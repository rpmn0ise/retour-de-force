## Description

La Toyota AE86 Trueno revisitée pour BeamNG.drive. Modèle haute-fidélité avec
physique entièrement retravaillée pour le drift. Suspension avant/arrière custom,
LSD Torsen simulé, ratio de boîte sport.

## Contenu du mod

- Modèle 3D haute qualité (LOD multiples)
- 3 livrées : blanc/noir Panda, rouge sport, Fujiwara Tofu
- Config suspension drift pré-installée
- Sons moteur 4A-GE remastérisés

## Installation

1. Télécharger le `.zip`
2. Ouvrir BeamNG → Repository local
3. Glisser le fichier dans la fenêtre
4. Redémarrer le jeu
