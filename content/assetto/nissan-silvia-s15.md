## Description

La Nissan Silvia S15 Spec-R au standard AC Community — la référence qualité
pour les mods Assetto Corsa. Physique développée en collaboration avec des pilotes
réels, données aéro issues de mesures en soufflerie.

## Spécifications

| Paramètre | Valeur |
|---|---|
| Moteur | SR20DET 2.0L Turbo |
| Puissance | 250 ch (stock) |
| Transmission | 6-vitesses manuelle |
| Poids | 1280 kg |
| Traction | RWD |

## Variantes incluses

- **Street** : Configuration route, pneumatiques Bridgestone
- **Drift** : LSD mechanial, angle kit, pneus semi-slick
- **Time Attack** : Aéro aggresive, pneus slick

## Installation

1. Extraire dans `Assetto Corsa/content/cars/`
2. Lancer Content Manager pour vérifier l'intégrité
3. Disponible dans la sélection voiture

## Compatibilité

- Assetto Corsa v1.16+
- Content Manager : ✓
- Custom Shaders Patch : Recommandé (ext_config inclus)
- Sol : Compatible
